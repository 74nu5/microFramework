# microFramework

Petit framework JavaScript expérimental (`js/micro.js`) explorant la manipulation du DOM, les collections (Enumerable, Array) et la sélection d'éléments via le moteur de sélecteurs [Sizzle](https://sizzlejs.com/). Le fichier `index.html` sert de page de test et le dossier `js/` contient le code ainsi que ses modules de test.

> Projet personnel ancien (2012), expérimental et non maintenu.

**Techno :** JavaScript, Sizzle
